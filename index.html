គ្រប់គ្រងប្រភេទ (Category Manager)

បន្ថែម / កែ / លុប ប្រភេទ. Categories are stored in Firestore and update in real-time.

Categories below update via Firestore onSnapshot. Use Edit / Delete next to each item.

Click Add to create a category. If duplicate names exist, the code prevents creating identical names.

សម្រាប់អ្នកលក់ — ជ្រើសប្រភេទ → ជ្រើសទំនិញ → បញ្ចូលចំនួន → លក់

Notes — repeated block 1

This project stores data in Firebase Firestore and Storage. Data persists online.

If you want offline-only behavior, remove Firebase and use localStorage instead.

Notes — repeated block 2

Make sure authentication is enabled: Firebase Console → Authentication → Sign-in method → Email/Password.

Create at least two accounts: admin@store.com and seller@store.com

Notes — repeated block 3

For image uploads, Firebase Storage rules must allow uploads by authenticated users or adjust rules in Console.

If you see permission errors, open Firebase Console → Storage → Rules and temporarily set to allow for testing.
